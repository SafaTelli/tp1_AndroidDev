    Overview

    L'objectif de ce cours est d'initier les étudiants aux développement sous Android. À la fin de ce cours, les étudiants doivent maitriser les notions suivantes :

    • Les composants graphiques sous Android.
    • La navigation et la communication entre les application Android.
    • La manipulation des données ( fichiers, BD interne ...).
    • La communication avec les services web.

    Ce cours comporte cinq séances de travaux pratiques:

    • TP1: Éléments Graphiques de base et Intents
    • TP2: Éléments Graphiques avancés : Adaptateurs & Listes
    • TP3: Éléments Graphiques avancés : Fragments & Menus
    • TP4: Base de donnés internes/ Firebase
    • TP5: Connexion à des services web REST : Retrofit
